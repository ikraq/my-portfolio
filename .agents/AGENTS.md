# Portfolio Auto-Push Rule

Whenever you (the AI assistant) make changes to any files in this workspace:
1. You MUST automatically stage all changes using `git add .`
2. You MUST commit the changes with a brief, descriptive message using `git commit -m "..."`
3. You MUST push the changes to the remote repository using `git push`

This ensures the user's live Vercel website is always up to date with the latest changes you make without them having to ask.
